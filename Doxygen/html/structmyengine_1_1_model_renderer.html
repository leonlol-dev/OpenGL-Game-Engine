Detailed Description

This is the model renderer.

It renders a model.

The model renderer has multiple variable needed to render which is the shader, vao, model, texture and optionally material. The user has to to set the model first as the model renderer requires a model object in the constructor. onDisplay() function is what displays the model using the shader and setting the uniforms to the shaders as well as binding the wanted texture. Users can change the shaders using material resource.
