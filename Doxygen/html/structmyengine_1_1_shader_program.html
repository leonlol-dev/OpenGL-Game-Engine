Detailed Description

This is the shader program.

It handles anything to do with the vertex shader and fragment shader of OpenGL.

The constructor of the shader program takes in the path of the vertex and fragment shader files, converts it into string and then attaches it into OpenGL. The shader program is also responsible for sending the uniform locations and functions to the shader via the void functions - this is done usually in the the model renderer.
